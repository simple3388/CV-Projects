This file will have a short description on the projects found in this repisotory.

- The Sealed Sanctuary

  This project is developed in Unity using C#
  
  The player is trapped inside an ancient temple and must push key stones onto specific tiles to unlock the door leading to the exit. Movement is restricted by walls and environmental hazards such as fire,
  which cannot be crossed and must be extinguished by pushing nearby water pots onto it in order to clear a path forward. Each Level also has an easy mode which will give the player the freedom to move
  an unlimited number of times and a hard mode which will have a move limit.


- KnightSurvivor

  This project is developed using RayLib using C++

  This is a Vampire Survivor type of game where the player must survive for a set amount of time while enemies approach from all sides. The weapons will auto attack the enemies that spwan and the player
  inputs will mainly be just the movement to try to avoid hitting enemies. Enemies drop xp chests which the player can pick up in order to level up and upgrade and choose an attribute and upgrade it.
  In order to match the players power increase the enemy spawn rate increases the longer he survives.
